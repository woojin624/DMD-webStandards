텍스트 관련 요소

강조 - strong, em

- 강한 강조 <strong> : 굵게 출력합니다.
- 강한 강조 <em> : 기울림꼴로 출력합니다.

strong태그로 출력한 글씨입니다.

추가글, 삭제글 - ins, del

- 추가글 <ins> : 텍스트의 아래쪽에 밑줄 표시합니다.
- 삭제글 <del> : 텍스트의 중간에 밑줄 표시합니다.

- 아랫첨자 <sub> : 텍스트의 오른쪽 아래에 작게 표시합니다.

ins태그로 출력한 글씨입니다.

del태그로 출력한 글씨입니다.


테이블 관련 요소

테이블 요소 - table, tr, th, td

선언 <table>
행 <tr>
- 제목 셀 <th>
- 내용 셀 <td>

셀 병합 - rowsapn, colspan

가로로 합치기 colsapn
세로로 합치기 colsapn

제목 및 요약문 - caption, summary

summary
- 간단한 제목보다 테이블의 내용을 설명합니다.
- 시각적으로 테이블 접근이 쉽지 않은 사용자를 위한 배려
caption
- 테이블의 내용을 대표할 수 있는 제목을 작성합니다.

행 그룹화 요소 - thead, tfoot, tbody

--선언 순서 : thead -> tfoot -> tbody
thead
- 테이블의 헤더 영역. 반드시 th요소를 사용하여 제목 셀이라는 것을 명시합니다.
tfoot
- 위치에 관계 없이 테이블의 가장 마지막 행에 출력합니다.
tbody
- 논리적 구분을 위해 여러 번 등장할 수도 있습니다.

제목 셀과 내용 셀의 연관성 - scope

- scope = "row" : 행의 제목
- scope = "col" : 열의 제목
